Chess Rook

The Chess Rook is part of a classic chess set, modeled in SolidWorks. This project highlights revolve-based geometry creation, fillets, and shelling to replicate an everyday object with aesthetic detailing.

Design Details

  • Software Used: SolidWorks
  • Features Applied: Revolved Boss/Base, Cut-Extrude, Fillet
  • Applications: Part of a chess set, demonstrates aesthetic + functional modeling
